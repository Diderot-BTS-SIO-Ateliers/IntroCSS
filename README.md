# Introduction aux feuilles de styles
Pour créer le dépôt qui contiendra votre travail, cliquez sur ce lien : https://classroom.github.com/a/fb971Z1y

Vous devez effectuer 2 exercices : pour le premier, l'énoncé est fourni dans un fichier pdf ; pour le second, vous devez compléter les trois fichiers _niveau1.css_, _niveau2.css_ et _niveau3.css_. Vous devez respecter les attendus présentés dans les vidéos.

Informations importantes :
- l'évaluation de ce TP portera EXCLUSIVEMENT sur le travail effectué pendant la séance, elle tiendra compte non seulement du rendu, mais aussi du nombre de commits produits, de leur régularité et de leurs messages, qui doivent être pertinents.

- si vous utilisez un outil de traduction automatique des pages dans votre navigateur, souvenez-vous que les noms des balises HTML et des attributs de ces balises doivent rester dans la langue d'origine. Par conséquent, pensez toujours à regarder la version originale du code, et non pas uniquement la version traduite.
