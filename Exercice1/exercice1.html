Apprendre HTML

Voici un petit exercice utilisant les principales fonctionnalités de HTML, préparé par Sophie NABITZ

Tout d'abord, on peut organiser son texte en paragraphe. Le navigateur affiche tout paragraphe précédé et suivi d'un espace vide. Les informations sont automatiquement organisées sur plusieurs lignes en fonction de la taille de l'écran.

Vous pouvez essayer     de taper un texte
 comme cela,         mais vous remarquez bien que :
Les espaces consécutifs répétés sont tout simplement affichés comme un seul espace. Et les sauts de ligne dans le texte original ne sont pas pris en compte.
Pour aller à la ligne, il faut explicitement utiliser une balise <br>

Vous pouvez évidemment ajouter des caractères spéciaux, par exemple ajouter un petit symbole © après mon nom, mais de toute façon cela ne me fera pas gagner d'€ ... Tous les caractères spéciaux peuvent être consultés à cette adresse : http://alexandre.alapetite.fr/doc-alex/alx_special.html

Les liens entre pages

Vers une page du même site

Créez une deuxième page dans un fichier autrePage.html, dont le titre est "Ma seconde page". Et maintenant faisons un lien vers cette autre page. N'oubliez pas de prévoir de revenir vers la page actuelle à partir de la page 1...

Vers une page d'un autre site

Essayez maintenant de définir ce lien qui vous permettra de rejoindre l'URL suivante : https://www.netcampus.fr. Il faut que cette page se charge dans un nouvel onglet de votre navigateur.

Enfin n'oubliez pas que vous pouvez toujours proposer de remonter au début de la page ... ou d'aller tout en bas !

La validation du HTML

Il est temps maintenant de passer à la validation de votre page HTML. Il faudra le faire régulièrement pour être sûr que votre document est bien formé, et que votre code source est valide. Vous pouvez consulter un validateur de code HTML en ligne, par exemple celui à l'adresse http://validator.w3.org/, il devra ne signaler aucune erreur.

Les tableaux de données

Étape suivante : mettons en forme des tableaux de données ...

Voici les données avec lesquelles nous allons travailler :

    4    5    6

Pour que les cases du tableau aient toutes une mise en forme convenable, ajoutez les lignes suivantes dans la balise head de votre page :

    <style>
        td {
            text-align: center;
            border: 1px solid;
            width: 50px;
        }
    </style>
